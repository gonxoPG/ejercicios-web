# ejercicios-web 💻

Selección de ejercicios prácticos realizados durante el Ciclo Formativo 
de Grado Superior en Desarrollo de Aplicaciones Web 
(Universidad Europea, 2023-2025).

El repositorio recoge los ejercicios más representativos de tres 
módulos, organizados por tecnología y temática.

---

## Estructura

### 📁 html-css
Ejercicios de diseño web y maquetación, incluyendo prácticas 
evaluables y ejercicios de CSS avanzado.

**Tecnologías:** HTML5 · CSS3

**Contenido destacado:**
- Ejercicios evaluables de diseño y maquetación
- Práctica de CSS avanzado (UT2.2)
- Prácticas de diseño de interfaces (UD1)

---

### 📁 php
Ejercicios de desarrollo backend en PHP, desde fundamentos 
hasta conexión con bases de datos.

**Tecnologías:** PHP · MySQL · PDO · MySQLi

**Contenido destacado:**
- Introducción y fundamentos de PHP
- Sesiones, cookies y funciones avanzadas
- Programación orientada a objetos (POO)
- Gestión de ficheros
- Conexión a base de datos con MySQLi y PDO

---

### 📁 html-css-php
Ejercicios que integran frontend y backend, incluyendo 
formularios con validación y conexión a base de datos.

**Tecnologías:** HTML5 · CSS3 · JavaScript · PHP · MySQL

**Contenido destacado:**
- Formulario de registro interactivo con validación
- Formulario conectado a base de datos
- Captcha implementado en PHP
- Ejercicios integrados HTML, CSS y JavaScript

---

## Contexto

Estos ejercicios forman parte de mi formación en DAW y 
complementan mi proyecto final [RGS Wonderlust](https://github.com/gonxoPG/rgs-wonderlust), 
una plataforma web completa desarrollada en equipo con 
HTML5, CSS3, JavaScript, PHP y MySQL.
